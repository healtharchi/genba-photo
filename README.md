# HEALTH ARCHI 現場写真（T008）

一人親方・小規模工務店向けの、超ミニマルな現場写真アプリです。

## このアプリの考え方
- 撮る前に「現場」と「工種」を選ぶ → あとは撮るだけ
- 撮った瞬間に名前（工種_日付_連番）が自動でつく
- 現場ごと・工種ごとに自動で整理される（後から整理する作業ゼロ）
- 写真は端末の中だけに保存。外部送信なし・ログインなし・難しい連携設定なし
- 会社のパソコンへ移したいときは「探す」画面の「写真アプリに保存」を押すだけ

## 構成
- index.html … アプリ本体

## 技術メモ
- React 18 + Tailwind（CDN）/ 単一HTMLファイル
- 写真の保存：IndexedDB（ブラウザ内の大容量保管庫）
- 現場名の保存：localStorage
- 書き出し：Web Share API（iPhone Safari）。非対応環境ではダウンロードで代替

## 注意
- カメラ撮影・「写真アプリに保存」は iPhone の Safari で動作します。
- PWA（ホーム画面に追加してアプリのように使う）対応。
